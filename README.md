## 概要
IchigoJam（1台）をブラウザ上から遠隔操作できます。

IchigoJamとサーバーとはTCPで接続します。

今回はできるだけシンプルなものにすべく、ブラウザ → IchigoJamの単方向のみの実装です。

IchigoJamとブラウザの双方向通信はこちらのアプリで実装しています。

https://github.com/shoichi1031da/ichigojam-e-remocon-app
